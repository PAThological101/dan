Hey Viv. Close but no cigar. The p-value must be less than (not including) 0.05 in order for the outcome to be considered statistically significant. As with any case in which a statistically significant finding was not obtained, the only inference that can be conveyed in the APS is that a statistical difference was not attained between the study groups.
